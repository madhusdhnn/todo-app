# todo-client

Todo client is React+Context-API app bundled with Webpack-v4 and babel

#### Prerequisite

* Node >= 8.15.1
* npm >= 6.4.1

#### Steps for project setup

* `npm install`
* `npm start` (for dev)
* `npm run build` (for production)

Press `Ctrl+C` or `CMD+C` (on MacOS) to interrupt `npm`
